This directory is dedicated to Homework Assignment #1 in the Software Engineering for Data Scientists
course taught at Unviersity of Washington, Winter Quarter, 2016.

The Data directory contains the raw data of bikes crossing the Fremont bridge since Oct. 2012 to present
downloaded from Seattle City website on January 10, 2016.
The Analysis directory contains the Analysis of the data requested for the homework assignment.
Namely: 
(1) Isolate the 2015 data, creating fields for date, hour, and count; 
(2) use python matplotlib to plot the counts by hour; 
(3) compute the hourly average counts; and 
(4) determine what is the busiest hour of the day.

This is to be done in an iPython notebook.
